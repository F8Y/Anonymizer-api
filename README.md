# Anonymizer API

REST-API-модуль на PHP для обезличивания персональных данных.

## О проекте

Данный проект представляет собой отдельный модуль, предоставляющий REST API для обезличивания
входных данных.

Основная задача модуля:
- принять структурированные данные в формате JSON;
- применить правила обезличивания;
- вернуть обезличенный результат в стандартизированном виде.

Проект не зависит от конкретной LMS, CMS или внешней платформы. 
HTTP-слой рассматривается как адаптер над модулем обелзичивания.

## Фунционал

### Базовый функционал

- прием JSON-запросов на обезличивание данных;
- валидация входных данных;
- применение набора правил обезличивания;
- возврат результата в формате JSON;
- обработка ошибок.

### Правила обезличивания

- скрытие или замена ФИО;
- маскирование адреса электронной почты;
- маскирование номера телефона;
- преобразование даты рождения в менее точное представление;
- обезличивание иных персональных полей по заданным правилам.

## Архитектура

- **Domain** — ядро бизнес-логики обезличивания;
- **Application** — сценарии использования;
- **Http** — REST API и обработка запросов;
- **Infrastructure** — технические детали, не относящиеся к предметной логике.

Такой подход позволяет:
- не смешивать HTTP и бизнес-логику;
- тестировать правила обезличивания отдельно;
- в будущем переиспользовать ядро в других окружениях.
